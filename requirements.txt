배포 전 준비사항
===============================================================================
1. 각 지갑에 devnet SOL 에어드롭 필요:
   solana airdrop 2 <deployer-address> --url https://api.devnet.solana.com
   solana airdrop 2 <job-seeker-address> --url https://api.devnet.solana.com
   solana airdrop 2 <recruiter-address> --url https://api.devnet.solana.com

2. 배포 명령어:
   solana config set --url https://api.devnet.solana.com
   solana config set --keypair deployer-wallet.json
   anchor build
   anchor deploy

3. 테스트 시나리오:
   - Deployer: 프로그램 배포 및 관리
   - Job Seeker: 프로필 생성, 잡 지원, 스카우트 응답
   - Recruiter: 잡 생성, 지원서 검토, 스카우트 발송
